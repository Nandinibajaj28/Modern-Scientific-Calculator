## 🚀 Features
- Basic arithmetic operations (+, −, ×, ÷)
- Advanced functions (sin, cos, tan, log, exponent, etc.)
- Toggle between **Basic** and **Scientific** modes
- Responsive design for desktop & mobile
- Keyboard input support
